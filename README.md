# Python URL Shorterner


## Installation

Pre-requisites:

- Python

Install dependencies, create the SQLite database and run the flask application

```
  pip install -r requirements.txt
  flask run
```

You can access the application at 127.0.0.1:5000 or <your_domain>:5000.
